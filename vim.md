# Vim shortcuts

## Replace all occurences of word

```
:%s/oldWord/newWord/g
```

## Copy/Cut/Paste
- `V` - Enter visual mode
  - `Ctrl + v` - Enter block (vertical) mode
  - `y` - copy (yank)
  - `d` - cut
- ´p´ - paste, in normal mode

## Tabs
- New tab
  ```
  :tabnew «filename»
  ```
- `Ctrl + w Shit + t` - Pane to new tab
- `gt` - go to next tab
